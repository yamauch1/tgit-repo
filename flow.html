◆Git開発入門開発

開発の流れ

  1. リモートリポジトリを作成

    2. GitHubを利用してリモートリポジトリを作成

  2. Gitアカウントをプロジェクトチームに招待

    3. GitHubでチームメンバーを招待し共有を有効化

  3. チーム招待を承認

    4. 参加者はGitHubアカウントにログインし、招待を承認

  4. Cloneを使いローカルリポジトリを作成

    5. ローカルリポジトリを作成し、ローカル環境を構築

  5. コードエディタでプロジェクトを開く (VsCode推奨)

    6. プロジェクトがローカル環境にあるかの確認

  6. 開発開始！

作業環境構築

  1. ブランチを作成

    2. ブランチを作成し作業場所を切り離す

  2. 変更の反映

    3. コミットすると現在のブランチに変更記録が残る

  3. マージ

    4. マージをすることで切り離したブランチの内容を本体に反映

  4. プッシュ

    5. ローカルリポジトリからリモートリポジトリに反映させ全体に共有

  5. プル

    6. 変更を加えられたリモートリポジトリの内容をローカルへ反映

  6. 最初に戻る
